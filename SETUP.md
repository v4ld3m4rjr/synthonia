## Setup SynthonIA

### 1. Pré-requisitos
- Node.js 18+
- Conta no Supabase
- Conta no GitHub

### 2. Instalação
```bash
npm install
npm run dev
```

### 3. Variáveis de Ambiente
Copie `.env.local.example` para `.env.local` e preencha com suas chaves.

### 4. Banco de dados
Importe `supabase/schema.sql` no Supabase.
